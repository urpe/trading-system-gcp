# Sistema de Trading Algorítmico v1.0

Sistema de trading serverless en Google Cloud Platform

## Componentes

| Servicio | Descripción |
| :--- | :--- |
| data-ingestor | Ingesta datos de Binance |
| strategy-agent | Genera señales de trading |
| order-agent | Ejecuta órdenes en Binance Testnet |
| dashboard | Monitoreo visual |
| alert-service | Notificaciones |
